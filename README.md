# 🚀 Toxilisk Messenger

**Полноценный мессенджер с админкой, каналами, голосовыми сообщениями, подарками и монетами**  
Работает в реальном времени через WebSocket. Стиль как в Telegram.

---

## ✨ Возможности

### 💬 Сообщения
- Мгновенная отправка (WebSocket)
- Текстовые сообщения
- Голосовые сообщения (запись через микрофон)
- Отображение времени отправки

### 👤 Пользователи
- Регистрация и вход
- Настройка профиля (имя, аватар)
- Система уровней и опыта
- Монеты (за активность и подарки)
- Роли: owner, admin, moderator, user, restricted

### 📢 Каналы
- Создание каналов
- Подписка на каналы
- Верификация каналов
- Премиум-каналы
- Накрутка подписчиков (админ)

### 🎁 Подарки
- 6 видов подарков (торт, цветы, мишка, драгоценность, игра, сердце)
- Покупка за монеты
- Отправка пользователям
- История полученных подарков

### 👑 Админ-панель (70+ функций)
- **Управление пользователями**: бан, разбан, мут, размут, варны, кик
- **Роли**: выдать админа, модератора, ограничить
- **Премиум**: выдать/снять
- **Верификация**: выдать/снять галочку
- **Экономика**: выдать/забрать монеты, опыт, изменить уровень
- **Каналы**: верификация, премиум, накрутка подписчиков (100/1000/10000), удаление
- **Глобальные**: очистка сообщений, полный сброс, глобальный мут, объявления
- **Экспорт**: выгрузить всех пользователей, все сообщения

### 📱 Интерфейс
- Тёмная тема (как в Telegram)
- Адаптивный дизайн
- Анимации сообщений
- Показ онлайн-статуса

## 🛠 Технологии

| Технология | Назначение |
|------------|------------|
| Node.js | Серверная платформа |
| Express | Веб-фреймворк |
| WebSocket (ws) | Реальное время |
| Multer | Загрузка файлов (голосовые) |
| JSON-файлы | Хранение данных (без БД) |
